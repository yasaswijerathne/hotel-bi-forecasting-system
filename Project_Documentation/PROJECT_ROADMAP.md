# Hotel Business Intelligence & Forecasting System

## Project Goal

Develop an end-to-end Hotel Business Intelligence and Forecasting Platform using PostgreSQL, Python, SQL, Power BI, and Data Warehousing.

---

## Development Phases

### Phase 1 - Operational Database
- Database Design
- Entity Relationship Diagram (ERD)
- PostgreSQL Implementation

### Phase 2 - Data Warehouse
- Fact Tables
- Dimension Tables
- Star Schema Design

### Phase 3 - ETL Development
- Data Extraction
- Data Transformation
- Data Loading

### Phase 4 - SQL Analytics
- Revenue Analysis
- Occupancy Analysis
- Customer Analytics
- Inventory Analytics

### Phase 5 - Power BI Dashboards
- Executive Dashboard
- Financial Dashboard
- Customer Dashboard
- Inventory Dashboard

### Phase 6 - Forecasting
- Revenue Forecasting
- Occupancy Forecasting
- Inventory Forecasting

### Phase 7 - Android Application
- Operational Interface
- Guest Management
- Reservation Management
